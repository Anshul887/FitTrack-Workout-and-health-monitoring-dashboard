# FitTrack Enterprise

AI-powered Workout & Health Monitoring Platform built using MERN Stack, TypeScript, Redux Toolkit, Firebase Authentication, and Docker.

---

# Features

## Authentication
- JWT Authentication
- Firebase Google Login
- Protected Routes
- Role-Based Access
- Admin Authentication

---

## Dashboard
- Health Analytics
- Workout Tracking
- Calories Monitoring
- Heart Rate Monitoring
- Sleep Analytics
- BMI Calculator

---

## Workouts
- Full CRUD Workouts
- Workout Analytics
- Exercise Tracking
- Progress Reports
- Workout History

---

## Meal Planner
- Daily Meal Plans
- Nutrition Tracking
- Calories Breakdown
- Protein / Carbs / Fats Analytics

---

## AI Features
- AI Workout Suggestions
- Smart Health Insights
- Personalized Recommendations

---

## Admin Panel
- User Management
- Workout Moderation
- Analytics Dashboard
- Platform Reports

---

## UI/UX
- Premium Modern UI
- Dark Mode
- Mobile Responsive
- Framer Motion Animations
- Recharts Analytics

---

## DevOps
- Docker Support
- Docker Compose
- Nginx
- Production Deployment
- Vercel + Render Ready
